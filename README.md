# 美加墨 2026 观赛指挥舱

一个更贴近 2026 FIFA World Cup Canada/Mexico/USA 的静态 H5 工具。

它不再做泛世界杯小游戏，而是围绕本届世界杯最独特的信息设计：

- 当前赛况快照：按 FOX Sports 积分榜整理 12 个小组、积分、净胜球和第三名晋级区。
- 晋级对阵图：用 48 -> 32 -> 16 -> 8 -> 4 -> 2 -> 冠军的漏斗图解释本届扩军赛制。
- Polymarket 预测图例：把冠军市场和各组头名市场做成概率条，作为群体预期参考。
- 三国联合主办：加拿大、墨西哥、美国。
- 16 座主办城市：用地图和城市卡片整理国家、场馆、时区、承办场次。
- 48 队 / 104 场：用关键节点说明扩军后的赛程结构。
- 北京时间观赛：选择城市和当地开球时间，自动换算为中国球迷能理解的观看窗口。
- 路线式看球：开幕三国线、太平洋上午场、墨西哥文化线、东海岸决赛线、中部大球场线。
- 本地关注清单：只用浏览器 `localStorage` 保存关注城市，不需要登录、服务器、数据库或云存储。

## 运行方式

这是一个纯静态项目，直接打开 `index.html` 即可运行，也可以部署到 GitHub Pages。

## 项目定位

这个项目想做成“本届世界杯的轻量观赛入口”。它不是抽象足球小游戏，而是把 2026 美加墨世界杯的真实结构变成可交互的信息产品：当前积分、暂时晋级区、第三名榜、市场预测、三国、16 城、时区、路线和关键赛程节点，帮助中国球迷快速理解这届世界杯正在发生什么，以及这届为什么和以往不一样。

## 数据说明

页面中的赛况和预测是静态快照，不会自动联网刷新。当前版本整理于 2026-06-20，参考：

- FOX Sports FIFA World Cup standings
- Polymarket World Cup props
- FIFA 2026 官方赛事信息
